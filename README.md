- **State Management**: Redux-Toolkit
- **Form Validation**: React-Hook-Form
- **Schema Validation**: Zod
- **Server**: Express
- **ORM**: Prisma
- **Typesafe API**: tRPC
- **Database**: PlanetScale
- **Deployment**: Vercel
